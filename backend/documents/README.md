# Sample Document Directory

This folder should contain your PDF and PPTX files for the RAG system.

## Instructions

1. Add your PDF and/or PPTX files to this folder
2. Restart the backend server to index the documents
3. The RAG system will automatically extract and index the content

## Supported Formats

- PDF (.pdf)
- PowerPoint (.pptx)

## Example Documents

You can add:
- Product manuals
- Company policies
- Technical documentation
- Research papers
- Training materials
- FAQs
- Any domain-specific documents

## Tips

- Keep file names descriptive
- Ensure documents are text-based (not scanned images)
- Larger documents will be automatically chunked
- The system will cite sources in responses
